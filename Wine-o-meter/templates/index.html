How to make your prediction?


  1. Make requests from a python script using Wine-o-Meter's API:


    First, enter you input as a list of 11 numbers corresponding to the wine's characteristics:
    input = [7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]

    You also can make multiple entries at once:
    input = [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [5.0, 0.98, 0.32, 18.9, 0.050, 75.0, 122.0, 0.401, 3.1, 0.21, 1.2]]

    Then enter:
    requests.post("{{base_url+pred_url[1:]}}", json={"input": input}).json()

    Your result should appear in the form:
    {'predict_1': '6', 'predict_2': '5'}
